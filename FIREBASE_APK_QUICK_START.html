🔥 Firebase + APK Build

Complete Setup in 6 Steps

✅ Project Updated!
Firebase dependencies added to your project
Ready to build APK

1Get google-services.json from Firebase

  1. Go to: https://console.firebase.google.com
  2. Select your UniSync project
  3. Click gear icon ⚙️ → Project Settings
  4. Go to "Your apps" section
  5. Find Android app → Click download icon
  6. Save google-services.json to your computer
⚠️ IMPORTANT: This is NOT the Service Account JSON
Download the google-services.json specifically for Android

2Place google-services.json in Project

Copy the downloaded file to this exact location:

unisync/android/app/google-services.json
Right-click google-services.json → Copy
Navigate to: unisync/android/app/
Right-click → Paste

3Update .env File

Open the .env file in project root

Add this line:

ONESIGNAL_APP_ID=your-app-id-from-onesignal

Replace with your actual OneSignal App ID from OneSignal dashboard

Save the file

4Clean & Get Dependencies

Open Terminal/Command Prompt in project root

flutter clean flutter pub get

Wait for it to complete...

5Build the APK

Run these commands in order:

dart run build_runner build --delete-conflicting-outputs flutter build apk --release

⏳ First build takes 5-10 minutes. Please wait...

✅ When complete, you'll see:
"✓ Built build/app/outputs/apk/release/app-release.apk"

6Find & Test Your APK

Your APK file is here:

build/app/outputs/apk/release/app-release.apk

To install on phone:

  1. Copy APK to your phone
  2. Tap to install
  3. Allow "Unknown sources" if prompted
  4. Launch app → Tap "Allow" for notifications
  5. Test by creating an announcement
✅ SUCCESS: You'll get a push notification!

Troubleshooting

❌ "Could not find google-services.json"

Check: Is file at android/app/google-services.json?
Run: flutter clean
Then: flutter pub get

❌ Build is very slow

First build is slow (downloads dependencies)
Wait 5-10 minutes
Next builds will be faster

❌ "Gradle build failed"

flutter clean
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter build apk --release
